# Agent 工作规则

## 受保护目录

`skills/` 目录是当前仓库中的受保护只读区域。

对所有在本项目中工作的 agent，必须遵守以下规则：

- 不得修改 `skills/` 下任何已有文件。
- 不得在 `skills/` 下新增任何文件或目录。
- 不得删除、重命名、移动或替换 `skills/` 下任何文件或目录。
- 不得运行任何可能改写 `skills/` 下内容的格式化、代码生成、迁移或脚本操作。
- 默认将 `skills/` 下所有代码与文档视为不可变内容。

## 例外策略

唯一允许的例外情况是：用户在当前会话中明确授权可以修改 `skills/` 内的内容。

如果某个任务需要触碰 `skills/`，必须先停止并向用户确认。
